# Save 2 Ride 🏍️

A savings tracker game for electric dirt bikes. Pick a bike from the garage, stack money toward its real price, and watch it reveal layer by layer — confetti when you hit the goal.

- 25 real bikes from 79BIKE, TUTTIO, RIDING'TIMES, and SUR-RON
- Per-bike savings progress, saved in your browser
- Currency switcher: USD / EUR / GBP / CAD
- Single HTML file, works offline

Open `index.html` in a browser, or visit the hosted page.
